
## EDEN - A Garry's Mod Roleplaying Framework

### Information
 - Everything in this framework is using the snake_case code style 


### What does this framework have?
 - Heavily optimized for large communites 
 - Inventory system with MYSQL Support
 - Included Modules (Can be disabled): 
   - Limb System
   - Food & Hunger 
   - Stamina
   - Skills
   - Economy (Taxes)
